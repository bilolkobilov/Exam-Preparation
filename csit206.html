4. Show that \((p \lor q) \equiv \lnot(\lnot p \land \lnot q)\) using logical equivalences.

This is one of De Morgan's Laws. We can prove it using logical equivalences:

Starting with the right-hand side: $$\lnot(\lnot p \land \lnot q)$$ Applying De Morgan's Law: $$\equiv \lnot(\lnot p) \lor \lnot(\lnot q)$$ Using the double negation law (\(\lnot\lnot p \equiv p\)): $$\equiv p \lor q$$ This is the left-hand side, so we have proven the equivalence.

Therefore, \((p \lor q) \equiv \lnot(\lnot p \land \lnot q)\).

6. Use the algebra of propositions to prove \(((p \Rightarrow q) \land (q \Rightarrow r)) \Rightarrow (p \Rightarrow r)\).

We need to prove that \(((p \Rightarrow q) \land (q \Rightarrow r)) \Rightarrow (p \Rightarrow r)\) is a tautology.

We'll use the following logical equivalences: \(p \Rightarrow q \equiv \lnot p \lor q\) \(\lnot(p \land q) \equiv \lnot p \lor \lnot q\) (De Morgan's Law)

Starting with the given proposition: $$((p \Rightarrow q) \land (q \Rightarrow r)) \Rightarrow (p \Rightarrow r)$$ Replace implications with disjunctions: $$\equiv ((\lnot p \lor q) \land (\lnot q \lor r)) \Rightarrow (\lnot p \lor r)$$ Replace implication with disjunction: $$\equiv \lnot((\lnot p \lor q) \land (\lnot q \lor r)) \lor (\lnot p \lor r)$$ Apply De Morgan's Law: $$\equiv (\lnot(\lnot p \lor q) \lor \lnot(\lnot q \lor r)) \lor (\lnot p \lor r)$$ Apply De Morgan's Law again: $$\equiv ((p \land \lnot q) \lor (q \land \lnot r)) \lor (\lnot p \lor r)$$ Distribute: $$\equiv (p \land \lnot q) \lor (q \land \lnot r) \lor \lnot p \lor r$$ Rearrange: $$\equiv (\lnot p \lor p \land \lnot q) \lor (r \lor q \land \lnot r)$$ Since \(p \lor \lnot p\) is always true $$\equiv T \lor (q \land \lnot r) \lor r$$ Since T or anything is true $$\equiv T$$ Since the result is a tautology, the original proposition is proven.

8. Show that the logical expression \((p \lor q) \land (p \Rightarrow r) \land (q \Rightarrow s)\) is equivalent to \((r \lor s)\).

This is not generally true. It's a more complex problem, and the given simplification is incorrect. A truth table would show they are not equivalent. Here's a simpler, correct approach to show they are *not* equivalent.

Let's consider a counterexample: If p is True, q is False, r is False, and s is True.

Then, \((p \lor q) \land (p \Rightarrow r) \land (q \Rightarrow s)\) \(= (True \lor False) \land (True \Rightarrow False) \land (False \Rightarrow True)\) \(= (True) \land (False) \land (True)\) \(= False\)

But, \((r \lor s) = (False \lor True) = True\)

Since the two expressions yield different truth values for the same assignment of variables, they are not equivalent.

10. Define logical entailment and use it to prove that \(p \land (p \Rightarrow q) \Rightarrow q\).

Logicalentailment (or logical consequence) means that a proposition \(P\) entails a proposition \(Q\) (written \(P \Rightarrow Q\)) if, in every interpretation in which \(P\) is true, \(Q\) is also true.

We want to prove \(p \land (p \Rightarrow q) \Rightarrow q\). This is Modus Ponens.

We can prove this using a truth table: 


| p | q | p ⇒ q | p ∧ (p ⇒ q) | p ∧ (p ⇒ q) ⇒ q |
|---|---|-------|-------------|-------------------|
| T | T | T     | T           | T                 |
| T | F | F     | F           | T                 |
| F | T | T     | F           | T                 |
| F | F | T     | F           | T                 |

Since the last column is always true, \(p \land (p \Rightarrow q) \Rightarrow q\) is a tautology. Therefore, \(p \land (p \Rightarrow q)\) logically entails \(q\).

11. Prove that the relation \(R = \{(1,1), (2,2), (1,2), (2,3)\}\) is not transitive.

A relation \(R\) is transitive if for all \(a\), \(b\), and \(c\), if \((a, b) \in R\) and \((b, c) \in R\), then \((a, c) \in R\).

In the given relation \(R = \{(1,1), (2,2), (1,2), (2,3)\}\), we have: \((1, 2) \in R\) and \((2, 3) \in R\) However, \((1, 3) \notin R\)

Since we found a case where the condition for transitivity is not met, the relation \(R\) is not transitive.

12. Determine whether the relation \(R = \{(a, b) \mid a \equiv b \pmod{4}\}\) is an equivalence relation.

A relation is an equivalence relation if it is reflexive, symmetric, and transitive.

Reflexive: For all \(a\), \(a \equiv a \pmod{4}\) since \(a - a = 0\) is divisible by 4. So, \(R\) is reflexive.

Symmetric: If \(a \equiv b \pmod{4}\), then \(a - b\) is divisible by 4. This means \(b - a = -(a - b)\) is also divisible by 4, so \(b \equiv a \pmod{4}\). So, \(R\) is symmetric.

Transitive: If \(a \equiv b \pmod{4}\) and \(b \equiv c \pmod{4}\), then \(a - b\) and \(b - c\) are divisible by 4. Their sum, \((a - b) + (b - c) = a - c\), is also divisible by 4. Therefore, \(a \equiv c \pmod{4}\). So, \(R\) is transitive.

Since \(R\) is reflexive, symmetric, and transitive, it is an equivalence relation.

14. Define a partially ordered set and give an example.

A partially ordered set (or poset) is a set with a binary relation that is reflexive, antisymmetric, and transitive.

Example: The set of integers \(\mathbb{Z}\) with the relation "divides" (\(\mid\)).

Reflexive: For any integer \(a\), \(a \mid a\).

Antisymmetric: For any integers \(a\) and \(b\), if \(a \mid b\) and \(b \mid a\), then \(a = b\).

Transitive: For any integers \(a\), \(b\), and \(c\), if \(a \mid b\) and \(b \mid c\), then \(a \mid c\).

Thus, \((\mathbb{Z}, \mid)\) is a partially ordered set.

16. Prove that every equivalence relation induces a partition of the underlying set and illustrate with a sample case.

Theorem: Every equivalence relation on a set A induces a partition of A.

Proof: Let R be an equivalence relation on A. For any a ∈ A, let [a] denote the equivalence class of a. We need to show that the set of all equivalence classes forms a partition of A. That is, we need to show: 1. The equivalence classes are non-empty. 2. The union of all equivalence classes is A. 3. Any two equivalence classes are either equal or disjoint.

1. Since R is reflexive, (a, a) ∈ R for all a ∈ A. Thus, a ∈ [a], and [a] is non-empty.

2. Since every a ∈ A belongs to some equivalence class [a], the union of all equivalence classes is A.

3. Let [a] and [b] be two equivalence classes. Suppose [a] ∩ [b] ≠ ∅. Let c ∈ [a] ∩ [b]. Then (a, c) ∈ R and (b, c) ∈ R. Since R is symmetric, (c, b) ∈ R. Since R is transitive, (a, b) ∈ R. Now, let x ∈ [a]. Then (a, x) ∈ R. Since (a, b) ∈ R and R is transitive, (b, x) ∈ R. Thus, x ∈ [b]. So [a] ⊆ [b]. Similarly, [b] ⊆ [a]. Hence, [a] = [b].

Therefore, the equivalence classes form a partition of A.

Sample Case: Let A = {1, 2, 3, 4} and R = {(1, 1), (2, 2), (3, 3), (4, 4), (1, 2), (2, 1), (3, 4), (4, 3)}. R is an equivalence relation. The equivalence classes are: [1] = [2] = {1, 2} [3] = [4] = {3, 4} The partition of A induced by R is {{1, 2}, {3, 4}}.

17. Show that \(\mathbb{N}\) under divisibility is a partially ordered set.

We need to show that the divisibility relation (\(\mid\)) on the set of natural numbers \(\mathbb{N}\) is reflexive, antisymmetric, and transitive.

Reflexive: For any \(a \in \mathbb{N}\), \(a \mid a\) since \(a = 1 \cdot a\).

Antisymmetric: For any \(a, b \in \mathbb{N}\), if \(a \mid b\) and \(b \mid a\), then \(a = kb\) and \(b = la\) for some integers \(k\) and \(l\). Substituting, we get \(a = k(la) = (kl)a\). Since \(a \neq 0\), \(kl = 1\). Since \(k\) and \(l\) are positive integers, \(k = l = 1\), so \(a = b\>.

Transitive: For any \(a, b, c \in \mathbb{N}\), if \(a \mid b\) and \(b \mid c\), then \(b = ma\) and \(c = nb\) for some integers \(m\) and \(n\). Substituting, we get \(c = n(ma) = (nm)a\), so \(a \mid c\).

Since divisibility on \(\mathbb{N}\) is reflexive, antisymmetric, and transitive, \((\mathbb{N}, \mid)\) is a partially ordered set.

19. Prove that the set of congruence classes modulo \(n\) forms a partition of \(\mathbb{Z}\).

Let \(\mathbb{Z}_n\) be the set of congruence classes modulo \(n\). We want to show that \(\mathbb{Z}_n\) forms a partition of \(\mathbb{Z}\).

For any integer \(a\), the congruence class \([a]_n\) is defined as \([a]_n = \{x \in \mathbb{Z} \mid x \equiv a \pmod{n}\}\).

We need to show that: 1. Each congruence class is non-empty. 2. The union of all congruence classes is \(\mathbb{Z}\). 3. Any two congruence classes are either equal or disjoint.

1. Each congruence class is non-empty since \(a \in [a]_n\) for all \(a \in \mathbb{Z}\) (since \(a \equiv a \pmod{n}\)).

2. The union of all congruence classes is \(\mathbb{Z}\) since every integer belongs to some congruence class.

3. Let \([a]_n\) and \([b]_n\) be two congruence classes. Suppose their intersection is not empty, and let \(c \in [a]_n \cap [b]_n\). Then \(c \equiv a \pmod{n}\) and \(c \equiv b \pmod{n}\). This implies \(a \equiv c \pmod{n}\) and \(c \equiv b \pmod{n}\). By transitivity, \(a \equiv b \pmod{n}\). Now, if \(x \in [a]_n\), then \(x \equiv a \pmod{n}\). By transitivity, \(x \equiv b \pmod{n}\), so \(x \in [b]_n\). Thus \([a]_n \subseteq [b]_n\). Similarly, \([b]_n \subseteq [a]_n\). Therefore, \([a]_n = [b]_n\).

Hence, the set of congruence classes modulo \(n\) forms a partition of \(\mathbb{Z}\).

21. Solve the recurrence relation \(a_n = 4a_{n-1} - 3a_{n-2}\) with \(a_0 = 1\), \(a_1 = 4\), showing all intermediate calculations.

The given recurrence relation is \(a_n = 4a_{n-1} - 3a_{n-2}\) with initial conditions \(a_0 = 1\) and \(a_1 = 4\).

The characteristic equation is \(r^2 - 4r + 3 = 0\). Factoring, we get \((r - 1)(r - 3) = 0\), so the roots are \(r_1 = 1\) and \(r_2 = 3\).

The general solution is \(a_n = A(1)^n + B(3)^n = A + B(3^n)\).

Using the initial conditions: For \(n = 0\): \(a_0 = 1 = A + B\) For \(n = 1\): \(a_1 = 4 = A + 3B\)

Solving this system of equations: Subtracting the first equation from the second: \(3 = 2B\), so \(B = \frac{3}{2}\) Substituting B into the first equation: \(1 = A + \frac{3}{2}\), so \(A = -\frac{1}{2}\)

The solution to the recurrence relation is: $$a_n = -\frac{1}{2} + \frac{3}{2}(3^n)$$

24. Prove that if \(f: A \rightarrow B\) and \(g: B \rightarrow C\) are bijections, then \(g \circ f\) is also a bijection.

To prove that \(g \circ f\) is a bijection, we need to show that it is both injective (one-to-one) and surjective (onto).

Injective: Let \(x_1, x_2 \in A\) and suppose \((g \circ f)(x_1) = (g \circ f)(x_2)\). Then \(g(f(x_1)) = g(f(x_2))\). Since g is injective, \(f(x_1) = f(x_2)\). Since f is injective, \(x_1 = x_2\). Therefore, \(g \circ f\) is injective.

Surjective: Let \(z \in C\). Since g is surjective, there exists \(y \in B\) such that \(g(y) = z\). Since f is surjective, there exists \(x \in A\) such that \(f(x) = y\). Thus, \(g(f(x)) = g(y) = z\). Therefore, for every \(z \in C\), there exists an \(x \in A\) such that \((g \circ f)(x) = z\). Hence, \(g \circ f\) is surjective.

Since \(g \circ f\) is both injective and surjective, it is a bijection.

25. Prove that any function from a finite set to itself that is injective is also surjective.

Let \(A\) be a finite set with \(n\) elements, and let \(f: A \rightarrow A\) be an injective function.

Since f is injective, for any distinct elements \(x_1, x_2 \in A\), \(f(x_1) \neq f(x_2)\). This means that f maps n distinct elements of A to n distinct elements in A.

The range of f, denoted as f(A), is a subset of A. Since f maps n distinct elements to n distinct elements, f(A) contains n elements.

Since A is finite and f(A) has the same number of elements as A, f(A) must be equal to A.

Therefore, for every element y in A, there exists an element x in A such that f(x) = y. This means f is surjective.

27. Show that every tree with \(n\) vertices has \(n - 1\) edges and provide a sample graph for verification.

Theorem: A tree with n vertices has n-1 edges.

Proof (by induction):

Base Case: n = 1. A tree with 1 vertex has 0 edges. The theorem holds.

Inductive Hypothesis: Assume that a tree with k vertices has k-1 edges for some k ≥ 1.

Inductive Step: Consider a tree T with k+1 vertices. Let e = (u,v) be an edge in T. Removing e disconnects T into two trees, T1 and T2, with k1 and k2 vertices, respectively. k1 + k2 = k + 1. By the inductive hypothesis, T1 has k1 - 1 edges and T2 has k2 - 1 edges. The total number of edges in T is (k1 - 1) + (k2 - 1) + 1 (for the edge e) = k1 + k2 - 1 = k + 1 - 1 = k. Thus, a tree with k+1 vertices has k edges.

By the principle of mathematical induction, the theorem holds for all n ≥ 1.

Consider a tree with 4 vertices: Vertices: {1, 2, 3, 4} Edges: {(1, 2), (2, 3), (3, 4)}

This tree has 4 vertices and 3 edges (4 - 1 = 3), verifying the theorem for this sample.

31. Prove that a graph is bipartite if and only if it contains no odd-length cycles.

Theorem: A graph is bipartite if and only if it contains no odd-length cycles.

Proof:

(⇒) Suppose G is bipartite with partitions U and V. Let C be a cycle in G. If C has length k, then we traverse k edges. Starting at a vertex in U, the path goes U -> V -> U -> V... If k is odd, the path ends in V, and the next edge goes to U. If the cycle ends at the starting vertex, the starting vertex must be in U. Thus the cycle must have even length. Therefore, if G is bipartite, it has no odd-length cycles.

(⇐) Suppose G has no odd-length cycles. If G is disconnected, we can consider each connected component separately. For a connected component, pick a vertex x and assign it to U. Perform a BFS. Assign vertices at even distance from x to U and vertices at odd distance from x to V. If there is an edge between two vertices in U, that would create an odd cycle. Similarly for V. Thus, if G has no odd-length cycles, G is bipartite.

32. Show that a connected graph with all vertices of even degree has an Eulerian circuit.

Theorem: A connected graph with all vertices of even degree has an Eulerian circuit.

Proof:

Start at any vertex v. Traverse the graph randomly, without repeating edges. Since every vertex has even degree, we can always exit a vertex we enter. The path must eventually return to v since we can't get stuck at a vertex with an odd degree. If this path includes all edges, it is an Eulerian circuit. If not, remove the edges of this circuit. The remaining graph still has all vertices of even degree. Repeat the process to find another circuit. Combine the circuits to form a larger circuit. Continue until all edges are included.
