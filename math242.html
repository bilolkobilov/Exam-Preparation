Module I: Bisection & Newton's Forward/Backward Method

The Bisection Method is a simple and robust numerical technique for finding the root of a continuous function. It works by repeatedly narrowing the interval in which a root is known to exist.

Steps of the Bisection Method:

  1. Choose initial guesses \( a \) and \( b \) such that \( f(a) \) and \( f(b) \) have opposite signs.
  2. Calculate the midpoint \( c = \frac{a + b}{2} \).
  3. Evaluate \( f(c) \).
  4. If \( f(c) = 0 \), then \( c \) is the root.
  5. If \( f(a) \) and \( f(c) \) have opposite signs, set \( b = c \); otherwise, set \( a = c \).
  6. Repeat steps 2-5 until the interval is sufficiently small or the desired accuracy is achieved.

Example 1:

Find the root of the equation \( f(x) = x^3 - 2x - 5 = 0 \) using the Bisection Method with initial guesses \( a = 2 \) and \( b = 3 \).

Solution:

Iteration 1:

\( a = 2 \), \( b = 3 \), \( f(2) = -1 \), \( f(3) = 16 \)

\( c = \frac{2 + 3}{2} = 2.5 \), \( f(2.5) = 5.625 \)

Since \( f(2) \) and \( f(2.5) \) have opposite signs, set \( b = 2.5 \)

Iteration 2:

\( a = 2 \), \( b = 2.5 \), \( f(2) = -1 \), \( f(2.5) = 5.625 \)

\( c = \frac{2 + 2.5}{2} = 2.25 \), \( f(2.25) = 1.890625 \)

Since \( f(2) \) and \( f(2.25) \) have opposite signs, set \( b = 2.25 \)

After several iterations, the root is approximately 2.09455

Example 2:

Find the root of the equation \( f(x) = x^2 - 4 = 0 \) using the Bisection Method with initial guesses \( a = 1 \) and \( b = 3 \).

Solution:

Iteration 1:

\( a = 1 \), \( b = 3 \), \( f(1) = -3 \), \( f(3) = 5 \)

\( c = \frac{1 + 3}{2} = 2 \), \( f(2) = 0 \)

Since \( f(2) = 0 \), 2 is the root.

Newton's Forward and Backward Interpolation methods are used to estimate the value of a function at an intermediate point within a given set of discrete points.

Newton’s Forward Difference Method

Newton's forward difference formula is used to interpolate values near the beginning of a data set.

The formula is: \[ P_n(x) = y_0 + \frac{u}{1!} \Delta y_0 + \frac{u(u-1)}{2!} \Delta^2 y_0 + \frac{u(u-1)(u-2)}{3!} \Delta^3 y_0 + \cdots \] where \( u = \frac{x - x_0}{h} \), \( h \) is the interval size, and \( \Delta y_0, \Delta^2 y_0, \Delta^3 y_0, ... \) are the forward differences.

Example 1:

Given the following data, find \( f(0.2) \) using Newton's Forward Difference formula:

x 0 0.1 0.2 0.3 0.4
f(x) 1 1.4 1.56 1.76 2

Solution:

First, we create the forward difference table:

x f(x) \( \Delta y \) \( \Delta^2 y \) \( \Delta^3 y \) \( \Delta^4 y \)
0 1 0.4 -0.24 0.2 0
0.1 1.4 0.16 -0.04 0.2
0.2 1.56 0.2 0.16
0.3 1.76 0.24
0.4 2

Here, \( x_0 = 0 \), \( h = 0.1 \), and \( x = 0.2 \). Therefore, \( u = \frac{0.2 - 0}{0.1} = 2 \).

Using Newton's Forward Difference formula: \[ f(0.2) = 1 + \frac{2}{1!}(0.4) + \frac{2(2-1)}{2!}(-0.24) + \frac{2(2-1)(2-2)}{3!}(0.2) \] \[ f(0.2) = 1 + 0.8 - 0.24 + 0 = 1.56 \]

Example 2:

Given the following data, find \( f(22) \) using Newton's Forward Difference formula:

x 20 25 30 35 40
f(x) 354 332 291 260 231

Solution:

First, we create the forward difference table:

x f(x) \( \Delta y \) \( \Delta^2 y \) \( \Delta^3 y \) \( \Delta^4 y \)
20 354 -22 -19 11 -23
25 332 -41 -8 -12
30 291 -31 3
35 260 -29
40 231

Here, \( x_0 = 20 \), \( h = 5 \), and \( x = 22 \). Therefore, \( u = \frac{22 - 20}{5} = 0.4 \).

Using Newton's Forward Difference formula: \[ f(22) = 354 + \frac{0.4}{1!}(-22) + \frac{0.4(0.4-1)}{2!}(-19) + \frac{0.4(0.4-1)(0.4-2)}{3!}(11) + \frac{0.4(0.4-1)(0.4-2)(0.4-3)}{4!}(-23) \] \[ f(22) = 354 - 8.8 + 2.28 + 0.8704 + 0.52416 = 348.87456 \]

Newton’s Backward Difference Method

Newton's backward difference formula is used to interpolate values near the end of a data set.

The formula is: \[ P_n(x) = y_n + \frac{v}{1!} \nabla y_n + \frac{v(v+1)}{2!} \nabla^2 y_n + \frac{v(v+1)(v+2)}{3!} \nabla^3 y_n + \cdots \] where \( v = \frac{x - x_n}{h} \), \( h \) is the interval size, and \( \nabla y_n, \nabla^2 y_n, \nabla^3 y_n, ... \) are the backward differences.

Example 1:

Given the following data, find \( f(25) \) using Newton's Backward Difference formula:

x 10 15 20 25 30
f(x) 1996 3992 8984 15976 25968

Solution:

First, we create the backward difference table:

x f(x) \( \nabla y \) \( \nabla^2 y \) \( \nabla^3 y \) \( \nabla^4 y \)
10 1996
15 3992 1996
20 8984 4992 2996
25 15976 6992 2000 -996
30 25968 9992 3000 1000 1996

Here, \( x_n = 30 \), \( h = 5 \), and \( x = 25 \). Therefore, \( v = \frac{25 - 30}{5} = -1 \).

Using Newton's Backward Difference formula: \[ f(25) = 25968 + \frac{-1}{1!}(9992) + \frac{-1(-1+1)}{2!}(3000) + \frac{-1(-1+1)(-1+2)}{3!}(1000) + \frac{-1(-1+1)(-1+2)(-1+3)}{4!}(1996) \] \[ f(25) = 25968 - 9992 + 0 - 0 + 0 = 15976 \]

Example 2:

Given the following data, find \( f(42) \) using Newton's Backward Difference formula:

x 20 25 30 35 40 45
f(x) 14.035 14.324 14.684 15.113 15.625 16.225

Solution:

First, we create the backward difference table:

x f(x) \( \nabla y \) \( \nabla^2 y \) \( \nabla^3 y \)\( \nabla^4 y \) \( \nabla^5 y \)
20 14.035
25 14.324 0.289
30 14.684 0.36 0.071
35 15.113 0.429 0.069 -0.002
40 15.625 0.512 0.083 0.014 0.016
45 16.225 0.6 0.088 0.005 -0.009 -0.025

Here, \( x_n = 40 \), \( h = 5 \), and \( x = 42 \). Therefore, \( v = \frac{42 - 40}{5} = 0.4 \).

Using Newton's Backward Difference formula: \[ f(42) = 15.625 + \frac{0.4}{1!}(0.512) + \frac{0.4(0.4+1)}{2!}(0.083) + \frac{0.4(0.4+1)(0.4+2)}{3!}(0.014) + \frac{0.4(0.4+1)(0.4+2)(0.4+3)}{4!}(0.016) + \frac{0.4(0.4+1)(0.4+2)(0.4+3)(0.4+4)}{5!}(-0.025) \] \[ f(42) = 15.625 + 0.2048 + 0.02324 + 0.0019584 + 0.0007168 + (-0.0003072) = 15.855418 \]

Module II: Numerical Differentiation & Integration

Numerical differentiation is the process of approximating the derivative of a function using numerical methods.

Forward Difference Formula

The forward difference formula is: \[ f'(x) \approx \frac{f(x + h) - f(x)}{h} \]

Backward Difference Formula

The backward difference formula is: \[ f'(x) \approx \frac{f(x) - f(x - h)}{h} \]

Central Difference Formula

The central difference formula is: \[ f'(x) \approx \frac{f(x + h) - f(x - h)}{2h} \]

Example 1:

Given the function \( f(x) = x^3 \), approximate the derivative at \( x = 2 \) using the central difference formula with \( h = 0.1 \).

Solution:

Using the central difference formula: \[ f'(2) \approx \frac{f(2 + 0.1) - f(2 - 0.1)}{2(0.1)} = \frac{f(2.1) - f(1.9)}{0.2} \] \[ f(2.1) = (2.1)^3 = 9.261 \] \[ f(1.9) = (1.9)^3 = 6.859 \] \[ f'(2) \approx \frac{9.261 - 6.859}{0.2} = \frac{2.402}{0.2} = 12.01 \]

The exact derivative of \( f(x) = x^3 \) is \( f'(x) = 3x^2 \), so \( f'(2) = 3(2)^2 = 12 \). The approximation error is |12.01 - 12| = 0.01

Example 2:

Given the function \( f(x) = sin(x) \), approximate the derivative at \( x = \frac{\pi}{4} \) using the forward difference formula with \( h = 0.01 \).

Solution:

Using the forward difference formula: \[ f'(\frac{\pi}{4}) \approx \frac{f(\frac{\pi}{4} + 0.01) - f(\frac{\pi}{4})}{0.01} \] \[ f'(\frac{\pi}{4}) \approx \frac{sin(\frac{\pi}{4} + 0.01) - sin(\frac{\pi}{4})}{0.01} \] \[ f'(\frac{\pi}{4}) \approx \frac{0.714121129 - 0.707106781}{0.01} \approx 0.7014348 \]

The exact derivative of \( f(x) = sin(x) \) is \( f'(x) = cos(x) \), so \( f'(\frac{\pi}{4}) = cos(\frac{\pi}{4}) = \frac{\sqrt{2}}{2} \approx 0.707106781 \).

Newton’s Method

Newton's method is an iterative method for finding the roots of a real-valued function. The formula is: \[ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} \]

Example 1:

Find the root of the equation \( f(x) = x^3 - 2x - 5 = 0 \) using Newton's method with an initial guess \( x_0 = 2 \).

Solution:

The derivative of \( f(x) \) is \( f'(x) = 3x^2 - 2 \).

Iteration 1:

\( x_1 = x_0 - \frac{f(x_0)}{f'(x_0)} = 2 - \frac{2^3 - 2(2) - 5}{3(2)^2 - 2} = 2 - \frac{-1}{10} = 2.1 \)

Iteration 2:

\( x_2 = x_1 - \frac{f(x_1)}{f'(x_1)} = 2.1 - \frac{2.1^3 - 2(2.1) - 5}{3(2.1)^2 - 2} = 2.1 - \frac{0.061}{11.23} \approx 2.0946 \)

The root is approximately 2.0946 after a few iterations.

Example 2:

Find the root of the equation \( f(x) = x^2 - 5 = 0 \) using Newton's method with an initial guess \( x_0 = 2 \).

Solution:

The derivative of \( f(x) \) is \( f'(x) = 2x \).

Iteration 1:

\( x_1 = x_0 - \frac{f(x_0)}{f'(x_0)} = 2 - \frac{2^2 - 5}{2(2)} = 2 - \frac{-1}{4} = 2.25 \)

Iteration 2:

\( x_2 = x_1 - \frac{f(x_1)}{f'(x_1)} = 2.25 - \frac{2.25^2 - 5}{2(2.25)} = 2.25 - \frac{0.0625}{4.5} \approx 2.23611 \)

The root is approximately 2.23611 after a few iterations. The actual root is \( \sqrt{5} \approx 2.236067977 \)

Numerical integration is the process of approximating the definite integral of a function using numerical methods.

Trapezoidal Rule

The Trapezoidal Rule approximates the integral by dividing the area under the curve into trapezoids. The formula is: \[ \int_a^b f(x) \, dx \approx \frac{h}{2} \left[ f(x_0) + 2f(x_1) + 2f(x_2) + \cdots + 2f(x_{n-1}) + f(x_n) \right] \] where \( h = \frac{b - a}{n} \) is the width of each subinterval, and \( n \) is the number of subintervals.

Example 1:

Approximate the integral \( \int_0^1 x^2 \, dx \) using the Trapezoidal Rule with \( n = 4 \).

Solution:

Here, \( a = 0 \), \( b = 1 \), and \( n = 4 \), so \( h = \frac{1 - 0}{4} = 0.25 \).

The values of \( x_i \) and \( f(x_i) \) are:

\( x_i \) 0 0.25 0.5 0.75 1
\( f(x_i) = x_i^2 \) 0 0.0625 0.25 0.5625 1

Using the Trapezoidal Rule: \[ \int_0^1 x^2 \, dx \approx \frac{0.25}{2} \left[ 0 + 2(0.0625) + 2(0.25) + 2(0.5625) + 1 \right] \] \[ \approx 0.125 \left[ 0 + 0.125 + 0.5 + 1.125 + 1 \right] = 0.125(2.75) = 0.34375 \]

The exact integral is \( \int_0^1 x^2 \, dx = \frac{1}{3} \approx 0.33333 \).

Example 2:

Approximate the integral \( \int_1^2 \frac{1}{x} \, dx \) using the Trapezoidal Rule with \( n = 5 \).

Solution:

Here, \( a = 1 \), \( b = 2 \), and \( n = 5 \), so \( h = \frac{2 - 1}{5} = 0.2 \).

The values of \( x_i \) and \( f(x_i) \) are:

\( x_i \) 1 1.2 1.4 1.6 1.8 2
\( f(x_i) = 1/x_i \) 1 0.83333 0.71429 0.625 0.55556 0.5

Using the Trapezoidal Rule: \[ \int_1^2 \frac{1}{x} \, dx \approx \frac{0.2}{2} \left[ 1 + 2(0.83333) + 2(0.71429) + 2(0.625) + 2(0.55556) + 0.5 \right] \] \[ \approx 0.1 \left[ 1 + 1.66666 + 1.42858 + 1.25 + 1.11112 + 0.5 \right] = 0.1(6.95636) \approx 0.695636 \]

The exact integral is \( \int_1^2 \frac{1}{x} \, dx = ln(2) \approx 0.69314718056 \)

Simpson’s 1/3 Rule

Simpson's 1/3 Rule approximates the integral by dividing the area under the curve into parabolic segments. The formula is: \[ \int_a^b f(x) \, dx \approx \frac{h}{3} \left[ f(x_0) + 4f(x_1) + 2f(x_2) + 4f(x_3) + 2f(x_4) + \cdots + 4f(x_{n-1}) + f(x_n) \right] \] where \( h = \frac{b - a}{n} \), \( n \) must be even.

Example 1:

Approximate the integral \( \int_0^2 x^4 \, dx \) using Simpson's 1/3 Rule with \( n = 4 \).

Solution:

Here, \( a = 0 \), \( b = 2 \), and \( n = 4 \), so \( h = \frac{2 - 0}{4} = 0.5 \).

The values of \( x_i \) and \( f(x_i) \) are:

\( x_i \) 0 0.5 1 1.5 2
\( f(x_i) = x_i^4 \) 0 0.0625 1 5.0625 16

Using Simpson's 1/3 Rule: \[ \int_0^2 x^4 \, dx \approx \frac{0.5}{3} \left[ 0 + 4(0.0625) + 2(1) + 4(5.0625) + 16 \right] \] \[ \approx \frac{0.5}{3} \left[ 0 + 0.25 + 2 + 20.25 + 16 \right] = \frac{0.5}{3}(38.5) \approx 6.41666666667 \]

The exact integral is \( \int_0^2 x^4 \, dx = \frac{2^5}{5} = \frac{32}{5} = 6.4 \).

Example 2:

Approximate the integral \( \int_1^3 ln(x) \, dx \) using Simpson's 1/3 Rule with \( n = 4 \).

Solution:

Here, \( a = 1 \), \( b = 3 \), and \( n = 4 \), so \( h = \frac{3 - 1}{4} = 0.5 \).

The values of \( x_i \) and \( f(x_i) \) are:

\( x_i \) 1 1.5 2 2.5 3
\( f(x_i) = ln(x_i) \) 0 0.405465 0.693147 0.916291 1.098612

Using Simpson's 1/3 Rule: \[ \int_1^3 ln(x) \, dx \approx \frac{0.5}{3} \left[ 0 + 4(0.405465) + 2(0.693147) + 4(0.916291) + 1.098612 \right] \] \[ \approx \frac{0.5}{3} \left[ 0 + 1.62186 + 1.386294 + 3.665164 + 1.098612 \right] = \frac{0.5}{3}(7.771926) \approx 1.295321 \]

The exact integral is \( \int_1^3 ln(x) \, dx = x*ln(x) - x \) evaluated from 1 to 3 which is \( 3*ln(3) - 3 - (1*ln(1) - 1) = 3*ln(3) - 2 \approx 1.295836866 \)

Simpson’s 3/8 Rule

Simpson's 3/8 Rule is another method for approximating definite integrals. The formula is: \[ \int_a^b f(x) \, dx \approx \frac{3h}{8} \left[ f(x_0) + 3f(x_1) + 3f(x_2) + 2f(x_3) + 3f(x_4) + 3f(x_5) + 2f(x_6) + \cdots + f(x_n) \right] \] where \( h = \frac{b - a}{n} \), and \( n \) must be a multiple of 3.

Example 1:

Approximate the integral \( \int_0^3 x^3 \, dx \) using Simpson's 3/8 Rule with \( n = 3 \).

Solution:

Here, \( a = 0 \), \( b = 3 \), and \( n = 3 \), so \( h = \frac{3 - 0}{3} = 1 \).

The values of \( x_i \) and \( f(x_i) \) are:

\( x_i \) 0 1 2 3
\( f(x_i) = x_i^3 \) 0 1 8 27

Using Simpson's 3/8 Rule: \[ \int_0^3 x^3 \, dx \approx \frac{3(1)}{8} \left[ 0 + 3(1) + 3(8) + 27 \right] \] \[ \approx \frac{3}{8} \left[ 0 + 3 + 24 + 27 \right] = \frac{3}{8}(54) = 20.25 \]

The exact integral is \( \int_0^3 x^3 \, dx = \frac{3^4}{4} = \frac{81}{4} = 20.25 \).

Example 2:

Approximate the integral \( \int_1^4 \sqrt{x} \, dx \) using Simpson's 3/8 Rule with \( n = 6 \).

Solution:

Here, \( a = 1 \), \( b = 4 \), and \( n = 6 \), so \( h = \frac{4 - 1}{6} = 0.5 \).

The values of \( x_i \) and \( f(x_i) \) are:

\( x_i \) 1 1.5 2 2.5 3 3.5 4
\( f(x_i) = \sqrt{x_i} \) 1 1.224745 1.414214 1.581139 1.732051 1.870829 2

Using Simpson's 3/8 Rule: \[ \int_1^4 \sqrt{x} \, dx \approx \frac{3(0.5)}{8} \left[ 1 + 3(1.224745) + 3(1.414214) + 2(1.581139) + 3(1.732051) + 3(1.870829)+ 2 \right] \] \[ \approx \frac{1.5}{8} \left[ 1 + 3.674235 + 4.242642 + 3.162278 + 5.196153 + 5.612487 + 2 \right] \] \[ \approx 0.1875(24.8878) \approx 4.6661 \]

The exact integral is \( \int_1^4 \sqrt{x} \, dx = \frac{2}{3}x^{3/2} \) evaluated from 1 to 4 which is \( \frac{2}{3}(4^{3/2} - 1^{3/2}) = \frac{2}{3}(8 - 1) = \frac{14}{3} \approx 4.66666666667 \)
